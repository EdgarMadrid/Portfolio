# ✨ About the Project

**Personal Portfolio** is a frontend project developed to create a modern and responsive portfolio website where projects, technical skills and professional information can be presented.

The project was created from a Figma design, applying professional frontend development practices focused on scalability, maintainability and clean code.

The main goal was not only to reproduce the visual design, but also to implement a structured frontend architecture similar to real-world projects.

---

# 🧩 Concepts Applied

During development, the following concepts were implemented:

✅ Semantic HTML5 structure  
✅ Component-based architecture  
✅ SCSS architecture following the **7-1 Pattern**  
✅ SCSS variables and configuration management  
✅ Reusable mixins  
✅ SCSS functions and formulas for dynamic values  
✅ BEM methodology for CSS naming  
✅ Mobile First workflow  
✅ Responsive design with media queries  
✅ SVG optimization and integration  
✅ Interactive hover states  
✅ Vite development workflow

# 🎨 SCSS Features

This project uses advanced SCSS organization techniques:

### Variables

Centralized management of:

- Colors
- Typography
- Spacing
- Breakpoints
- Global configuration


### Mixins

Reusable styles for:

- Media queries
- Responsive behavior
- Common layouts


### Functions

Dynamic calculations for:

- REM conversions
- Flexible sizing
- Consistent measurements


### 7-1 Architecture

SCSS organized following a scalable methodology:

- Abstracts
- Base
- Components
- Layout
- Pages
- Themes
- Vendors

---

# 📱 Responsive Design

The project follows a **Mobile First approach**:

Development starts from small screens and progressively adapts using media queries for:

- Mobile devices
- Tablets
- Desktop screens

# 🛠️ Run Locally

Clone repository:

```bash
git clone https://github.com/EdgarMadrid/Portfolio.git
```

Enter project:

```bash
cd Portfolio
```

Install dependencies:

```bash
pnpm install
```

Run development server:

```bash
pnpm run dev
```

Build project:

```bash
pnpm run build
```
